Welcome to my blog! This is the first post on my new personal website.

## What This Site Is

I built this site as a central place for my professional presence online — a home for my [resume](/resume/), [publications](/publications/), and thoughts on software engineering, research, and technology.

## What I'll Write About

I plan to write about:

- **Software engineering** — architecture decisions, interesting problems, tools I find useful
- **Research** — summaries of my work, interesting papers I've read, thoughts on the field
- **Technology** — broader trends, opinions, things I'm experimenting with
- **Lessons learned** — mistakes, debugging war stories, things I wish I'd known earlier

## The Stack

This site is built with [Hugo](https://gohugo.io/) using the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme, hosted on [GitHub Pages](https://pages.github.com/). Posts are written in Markdown — no CMS, no database, just files in a git repo.

If you want to get in touch, find me on [GitHub](https://github.com/Revys17) or drop me an email.

Thanks for stopping by. More posts coming soon.
